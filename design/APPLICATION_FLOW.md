## Cookie Consent Handling Flow

```
+---------------------------------------------------------------------------------+
|                        COOKIE CONSENT DISMISSAL                                  |
+---------------------------------------------------------------------------------+
|                                                                                  |
|  +-----------------+                                                             |
|  |  Page Loaded    |                                                             |
|  +-----------------+                                                             |
|          |                                                                       |
|          v                                                                       |
|  +-------------------+                                                           |
|  | dismissCookieConsent                                                          |
|  | enabled?          |----NO--->  Skip, proceed to extraction                    |
|  +-------------------+                                                           |
|          | YES                                                                   |
|          v                                                                       |
|  +---------------------------+                                                   |
|  |    TRY KNOWN CMPs         |  (Max 1s per selector)                            |
|  +---------------------------+                                                   |
|  | OneTrust    | Cookiebot   |                                                   |
|  | Usercentrics| TrustArc    |                                                   |
|  | Quantcast   | Didomi      |                                                   |
|  | Osano       | ConsentMgr  |                                                   |
|  +---------------------------+                                                   |
|          |                                                                       |
|          +-----> CMP Found?                                                      |
|          |         |                                                             |
|          |        YES --> Click accept --> Return {dismissed:true, mode:'cmp'}   |
|          |         |                                                             |
|          v        NO                                                             |
|  +---------------------------+                                                   |
|  |  TRY HEURISTIC            |                                                   |
|  +---------------------------+                                                   |
|  | Search for buttons with:  |                                                   |
|  | - "accept"  - "allow"     |                                                   |
|  | - "agree"   - "ok"        |                                                   |
|  | - "got it"  - "continue"  |                                                   |
|  +---------------------------+                                                   |
|          |                                                                       |
|          +-----> Button Found?                                                   |
|                    |                                                             |
|                   YES --> Click --> Return {dismissed:true, mode:'heuristic'}    |
|                    |                                                             |
|                   NO --> Return {dismissed:false, mode:'none'}                   |
|                                                                                  |
+---------------------------------------------------------------------------------+
|  IMPORTANT: Cookie dismissal is BEST-EFFORT.                                     |
|  Failures are logged but never block page loading.                               |
|  Total timeout: Max 3 seconds (1s per selector x 3 attempts)                     |
+---------------------------------------------------------------------------------+
```
